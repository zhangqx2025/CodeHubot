# 设备配置信息页面使用说明

## 📱 页面位置

**文件路径**: `/docs/kids-tutorial/device-info.html`

**访问方式**:
- 从首页 `index.html` 点击"设备配置信息"卡片进入
- 直接访问: `https://your-domain.com/docs/kids-tutorial/device-info.html`
- URL参数: `device-info.html?device=01` （自动选中设备01）

## 🎯 页面功能

### 1. 设备UUID查询
- 学生选择自己的设备编号（01-20）
- 显示对应的设备UUID
- 一键复制UUID功能

### 2. 预设指令说明
展示4个预设指令的详细信息：

| 指令名称 | preset_key | 颜色 | 用途 |
|---------|-----------|------|------|
| LED1打开3秒关闭 | led_seq_mig4wqy5 | 绿灯 | 可回收垃圾 |
| LED2打开3秒关闭 | led_seq_mig5hiep | 红灯 | 有害垃圾 |
| LED3打开3秒关闭 | led_seq_mig5i461 | 黄灯 | 厨余垃圾 |
| LED4打开3秒关闭 | led_seq_mig5ivu5 | 灰灯 | 其他垃圾 |

### 3. 配置步骤指导
- 复制UUID的步骤说明
- 创建智能体的步骤
- 绑定设备的方法
- 测试验证的方式

## 👨‍🏫 教学使用

### 课前准备

1. **分配设备编号**
   ```
   学生姓名 → 设备编号（01-20）
   ```

2. **投影展示**
   - 课堂上投影 `device-info.html`
   - 演示如何查询UUID
   - 演示如何复制UUID

3. **分发清单（可选）**
   - 打印 `/SQL/教学设备UUID清单.md`
   - 或使用 `/SQL/教学设备UUID列表.csv`

### 课堂流程

**第一步：查询设备信息（5分钟）**
1. 学生打开 `device-info.html`
2. 在下拉框中选择自己的设备编号
3. 记录或复制UUID

**第二步：创建智能体（15分钟）**
1. 进入系统创建智能体
2. 选择模板（猪猪侠/LED温湿度/垃圾分类）
3. 复制提示词

**第三步：绑定设备（5分钟）**
1. 在插件配置中
2. 找到自己的UUID对应的设备
3. 完成绑定

**第四步：测试验证（10分钟）**
1. 发送测试指令
2. 观察LED灯反应
3. 调整优化

## 🔧 维护更新

### 更新设备UUID

如果需要更新设备UUID，编辑 `device-info.html` 中的 `devices` 对象：

```javascript
const devices = {
    "01": { uuid: "18078072-cb2f-11f0-9ba7-00163e3c72a3" },
    "02": { uuid: "180788d2-cb2f-11f0-9ba7-00163e3c72a3" },
    // ...
};
```

### 更新预设指令

如果预设指令的 `preset_key` 发生变化，需要同步更新：
1. 页面中的预设指令卡片
2. 提示词示例部分

## ⚠️ 常见问题

### Q1: 学生忘记设备编号怎么办？
**A**: 教师维护一份 `学生-设备编号` 对照表

### Q2: UUID复制失败？
**A**: 页面提供了备用复制方法（手动选择复制），适配老版本浏览器

### Q3: 设备编号和UUID不匹配？
**A**: 核对 `/SQL/教学设备UUID清单.md` 中的记录

### Q4: 学生绑定了错误的设备？
**A**: 
1. 在系统中解绑错误设备
2. 重新查看 `device-info.html` 确认正确UUID
3. 重新绑定

## 🎓 教学技巧

### 技巧1：使用URL参数
为每个学生创建专属链接：
```
device-info.html?device=01  （学生A）
device-info.html?device=02  （学生B）
```

### 技巧2：截图对照
让学生截图保存自己的设备信息页面

### 技巧3：分组教学
- 第1组（设备01-05）
- 第2组（设备06-10）
- 第3组（设备11-15）
- 第4组（设备16-20）

### 技巧4：颜色记忆
- 绿灯 = 可回收（记：绿色环保）
- 红灯 = 有害垃圾（记：红色危险）
- 黄灯 = 厨余垃圾（记：黄色食物）
- 灰灯 = 其他垃圾（记：灰色其他）
