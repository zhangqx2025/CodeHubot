❓ 遇到问题了？

常见问题：

🤔 猪猪侠回答很冷淡怎么办？

检查一下系统提示词是不是复制对了，特别是"活泼开朗"这些描述性格的词。

🤔 垃圾分类小助手的灯不亮？

可能的原因：
1. 预设指令没有创建
2. preset_key名字不对
3. 设备没有上线
4. Temperature设置太高了（要改成0.2）

🤔 有时候灯亮，有时候不亮？

这说明Temperature设置太高了！让老师或爸爸妈妈把Temperature改成0.2，这样每次都会亮。
