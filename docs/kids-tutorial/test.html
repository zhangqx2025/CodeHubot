❓ 常见问题

🤔 猪猪侠回答很冷淡？

检查系统提示词是不是完整复制了，特别是"活泼开朗"这些性格描述。

🤔 LED灯不亮？

可能原因：
1. 预设指令没创建
2. preset_key名字不对
3. 设备没上线
4. Temperature设置太高（要改成0.2）

🤔 有时灯亮，有时不亮？

Temperature设置太高了！让老师把Temperature改成0.2，这样每次都会稳定工作。
