🔰 基础问题

什么是智能体？

智能体就像一个聪明的机器人朋友，你可以教它做不同的事情。比如聊天、回答问题、控制设备等。

在哪里创建智能体？

1. 打开网站：https://codehubot.hello1023.com/
2. 注册并登录
3. 找到左侧菜单的"智能体管理"
4. 点击"创建智能体"

我可以创建几个智能体？

想创建多少个都可以！每个可以有不同的功能。

🐷 猪猪侠问题

猪猪侠回答太冷淡？

原因：系统提示词没完整复制，漏掉了性格描述。

✅ 解决：重新复制系统提示词，确保"活泼开朗"、"表情符号"这些内容都有。

猪猪侠总是说一大堆话？

原因：AI没注意到"回复50-100字"的要求。

✅ 解决：在系统提示词最前面加上：
【重要】回答要简短，不超过100个字！

🗑️ 垃圾分类问题

LED灯完全不亮？

按顺序检查：

  • 设备在线吗？（让老师检查）
  • 预设指令创建了吗？（需要4个）
  • preset_key对吗？（名字要一模一样）
  • 选了插件吗？（要勾选"IoT设备控制"）

有时灯亮，有时不亮？

最常见问题！

⚠️ 可能原因：
  • 设备网络不稳定
  • 插件绑定不正确
  • 预设指令配置有误
✅ 解决方法：
  • 检查设备是否稳定在线
  • 重新绑定插件
  • 让老师检查预设指令配置

灯的颜色不对？

原因：预设指令绑定的LED号码不对。

正确的应该是：

  • LED1 = 绿灯（可回收）
  • LED2 = 红灯（有害）
  • LED3 = 黄灯（厨余）
  • LED4 = 灰灯（其他）
✅ 解决：让老师检查预设指令，确保每个预设控制的LED号码正确。

💡 实用技巧

让AI更聪明的秘诀

  • 清楚地说话 - 告诉AI要做什么
  • 多给例子 - 在系统提示词里写几个对话例子
  • 设规矩 - 用【禁止】告诉AI不要做什么
  • 多测试 - 创建后多试几次，有问题再改

🔒 安全提示

⚠️ 记住：
  • 不要告诉AI你的真实姓名、地址、学校
  • 不要把账号密码告诉别人
  • 遇到不懂的要问老师或爸爸妈妈
  • AI有时也会说错，重要的事要核实
