📚 AI智能体创建教学设计

适用年级：小学 3-6 年级 | 课时：2-3 课时
教学形式：项目式学习 + 实践操作

一、教学目标

🎯 知识与技能

  • 理解AI和智能体概念
  • 学会创建简单智能体
  • 掌握系统提示词编写
  • 了解工具调用原理

🧠 过程与方法

  • 培养逻辑思维能力
  • 学会问题调试方法
  • 体验完整项目流程
  • 掌握测试验证方法

❤️ 情感态度

  • 激发AI技术兴趣
  • 培养环保意识
  • 建立正确AI观念
  • 增强创新精神

二、教学重难点

类别 内容 突破策略
重点 系统提示词作用、为AI制定规则、测试验证 类比法、示例演示、即时反馈
难点 Temperature参数影响、preset_key对应、工具调用稳定性 对比展示、详细清单、教师辅导

三、教学准备

💻 系统环境：后端(8000)、插件服务(9000)、插件后端(9002)、MQTT、至少1台ESP32在线
📝 学生账号：每人独立账号（username: student01...）、密码简单易记
📋 材料准备：设备UUID清单、预设指令配置(JSON)、测试用例表格

四、教学过程

第一课时：创建猪猪侠（40分钟）

5分钟
🎬 导入
展示ChatGPT对话视频 → 提问"什么是AI" → 引入今天要创建AI助手
10分钟
📖 讲解
什么是智能体、系统提示词、AI怎么学习（用类比法：教新朋友聊天）
20分钟
👨‍💻 实践
教师演示(5分钟) → 学生操作(10分钟) → 测试对话(5分钟)
5分钟
📝 总结
提问总结、分享成果、预告下节课

第二课时：垃圾分类助手（45分钟）

5分钟
🔄 复习
回顾猪猪侠 → 引入垃圾分类（能控制LED灯）
10分钟
🗑️ 垃圾分类
分类游戏（实物展示） → 讲解4种垃圾和LED颜色对应
5分钟
🔌 插件和预设
什么是插件、预设指令、Temperature作用（0.2 vs 0.7对比）
20分钟
👨‍💻 实践
教师演示(3分钟) → 学生操作(12分钟) → 测试验证(5分钟)
5分钟
🎉 展示
学生演示 → 观察LED → 总结提升（技术让生活更美好）

五、评价标准（100分）

评价项 内容 分值
操作技能 独立完成猪猪侠(15分) + 辅助完成垃圾分类(15分) 30分
理解能力 理解提示词(10分) + 垃圾分类(10分) + 工具调用(10分) 30分
测试验证 完成测试(10分) + 准确记录(10分) 20分
创新分享 内容创意(10分) + 积极分享(10分) 20分
评价原则：重过程轻结果、鼓励尝试、差异化评价、即时反馈

六、课程总结

🎤 教师总结发言（3分钟）

学到了什么：通过这节课，我们不仅学会了创建AI智能体，更重要的是理解了AI如何学习。就像教新朋友说话一样，通过清晰的指令、具体的例子、明确的规则，AI就能按我们的期望工作。

技术与生活：垃圾分类小助手体现了技术如何服务生活、保护环境。当LED灯亮起的那一刻，看到的是AI、物联网、环保理念的完美结合。

培养的能力：逻辑思维能力、问题解决能力、创新实践能力，这些将是你们未来成长中宝贵的财富。

为所有认真学习、勇于尝试的同学们点赞！你们都是最棒的小小程序员！👏

八、常见问题应对

问题 原因 解决方案
学生操作速度差异大 计算机基础不同 快的帮慢的，准备进阶任务
LED灯不亮 预设或参数问题 教师提前测试，准备备用配置
学生不理解概念 讲解过于抽象 多用类比，少用技术术语
有时灯亮有时不亮 Temperature太高 对比演示0.7和0.2的差异
