🎓 什么是AI智能体？

🤖

像朋友一样

智能体就像一个聪明的机器人朋友，你可以教它说话、做事情！

📝

通过例子学习

就像教小朋友一样，给它看例子，告诉它怎么做，它就学会了！

⚙️

能做很多事

可以聊天、回答问题、控制设备，甚至帮你分类垃圾！

💡 简单理解：

想象你有一个很听话的机器人朋友，你告诉它：
"当别人说你好，你要友好地回应！"
"看到可回收垃圾，就打开绿色的灯！"
它就会按照你说的去做。这就是智能体！

🔑 三个重要概念

1️⃣ 系统提示词（最重要！）

这是什么？
就像给AI写的"说明书"，告诉它应该怎么做事、怎么说话。

类比理解：
就像你和新朋友第一次见面时说："我喜欢开玩笑，说话轻松一点。" 这样朋友就知道该怎么和你聊天了！

2️⃣ 插件（让AI有超能力）

这是什么？
给AI装上"技能包"，让它不仅能说话，还能做事情！

类比理解：
就像游戏里的装备，装上"IoT设备控制"这个插件，AI就能控制LED灯了！

3️⃣ 预设指令（提前设好的动作）

这是什么？
提前设置好的一系列动作，AI一个指令就能全部执行。

类比理解：
就像遥控器上的"一键回家模式"：按一下，灯亮、空调开、窗帘关，全都自动完成！
