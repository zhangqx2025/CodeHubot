🔑 核心概念

1️⃣ 系统提示词（最重要）

就像给AI写的"说明书"，告诉它怎么做事、怎么说话。
类比：就像告诉新朋友"我喜欢开玩笑，说话轻松一点"。

2️⃣ 插件

给AI装上"技能包"，让它能做更多事情。
类比：就像游戏里的装备，装上就有新能力。

3️⃣ 预设指令

提前设置好的一系列动作，一个指令就能全部执行。
类比：就像遥控器的"一键回家模式"。
