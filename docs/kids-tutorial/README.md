# 小学生AI智能体创建教程

专为小学生设计的AI智能体创建教程，简洁易懂。

## 🌐 系统地址

**CodeHubot 系统：** https://codehubot.hello1023.com/

- 注册账号（只需用户名+密码）
- 创建和管理智能体
- 测试对话功能
- 控制物联网设备

## 📂 文件说明

| 文件 | 说明 | 难度 |
|------|------|------|
| `index.html` | 首页（导航和概念讲解） | 全年级 |
| `device-info.html` | 🔥 设备配置信息查询（UUID和预设指令） | 全年级 |
| `zhuzhu.html` | 猪猪侠聊天助手教程 | ⭐ 简单 |
| `led-temp.html` | LED温湿度助手教程 | ⭐⭐ 中等 |
| `garbage.html` | 垃圾分类助手教程 | ⭐⭐ 中等 |
| `test.html` | 测试验证页面 | 全年级 |
| `help.html` | 帮助中心 | 全年级 |
| `教学设计.html` | 教师教学设计文档 | 教师用 |

## 🎯 三个项目

### 1. 猪猪侠聊天助手 🐷
- **类型**：闲聊型
- **难度**：⭐ 简单
- **时间**：15-20分钟
- **适合**：3-6年级

### 2. LED温湿度助手 💡
- **类型**：设备控制型
- **难度**：⭐⭐ 中等
- **时间**：20-30分钟
- **适合**：4-6年级

### 3. 垃圾分类小助手 🗑️
- **类型**：复杂应用型
- **难度**：⭐⭐ 中等
- **时间**：30-40分钟
- **适合**：4-6年级（需要老师辅助）

## 🚀 使用方法

### 学生使用
1. 用浏览器打开 `index.html`
2. **首先**：点击"设备配置信息"，查看自己的设备UUID
3. 点击"进入系统"按钮
4. 注册并登录
5. 跟着教程一步步操作，创建智能体时绑定对应设备

### 教师使用
1. 查看 `教学设计.html` 准备教学
2. 为每个学生分配设备编号（01-20）
3. 让学生访问 `device-info.html` 查看设备UUID
4. 指导学生操作
5. 辅助解决问题

## 💡 教学建议

**课时安排：**
- 第1课时：猪猪侠（20分钟）
- 第2课时：LED温湿度（30分钟）
- 第3课时：垃圾分类（40分钟）
- 第4课时：测试和分享（20分钟）

**重点辅导：**
- 设备UUID的查询和复制（必须第一步）
- 系统提示词的复制
- 插件的正确绑定
- 预设指令的配置

## 🎓 核心概念

- **系统提示词**：给AI写的"说明书"
- **插件**：给AI装的"技能包"
- **预设指令**：提前设置的一系列动作

## 📱 技术要求

- **浏览器**：Chrome/Edge/Firefox（推荐Chrome）
- **屏幕**：1024x768以上
- **网络**：稳定的互联网连接
