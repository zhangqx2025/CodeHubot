# 小学生智能体创建教程

这是一套专为小学生设计的AI智能体创建教程，采用HTML交互式页面形式，语言简单易懂，界面活泼有趣。

## 📂 文件说明

| 文件 | 说明 | 适合年级 |
|------|------|---------|
| `index.html` | 教程首页，导航页面 | 全年级 |
| `zhuzhu.html` | 猪猪侠聊天助手创建教程 | 3-6年级 |
| `garbage.html` | 垃圾分类小助手创建教程 | 4-6年级 |
| `test.html` | 测试和验证页面 | 全年级 |
| `help.html` | 帮助中心和常见问题 | 全年级 |

## 👨‍👩‍👧 给家长和老师的建议

### 教学准备

1. **提前准备**
   - 确保后端服务运行正常
   - 为学生创建好账号
   - 准备好设备UUID
   - 对于垃圾分类助手，提前创建好预设指令

2. **分组教学**
   - 3-4年级：建议只做猪猪侠聊天助手
   - 5-6年级：可以挑战垃圾分类助手
   - 技术较好的学生：可以尝试自定义

3. **时间安排**
   - 猪猪侠：15-20分钟
   - 垃圾分类：30-40分钟
   - 测试：10-15分钟

### 辅导要点

#### 猪猪侠聊天助手
- ✅ 重点：让学生理解"系统提示词"的作用
- ✅ 难点：复制粘贴提示词（可能需要帮助）
- ✅ 趣味点：测试对话，看AI如何回复

#### 垃圾分类助手
- ✅ 重点：理解垃圾分类规则
- ✅ 难点：
  - 预设指令创建（需要老师提前准备）
  - Temperature参数设置（需要老师协助）
  - preset_key的对应关系
- ✅ 趣味点：看LED灯亮起，直观感受AI控制硬件

### 需要老师/家长协助的环节

| 环节 | 协助内容 | 原因 |
|------|---------|------|
| **登录** | 输入账号密码 | 安全考虑 |
| **复制提示词** | 确保完整复制 | 避免遗漏 |
| **Temperature设置** | 修改为0.2 | 技术参数 |
| **预设指令创建** | 创建4个预设 | 较复杂的JSON配置 |
| **测试验证** | 检查LED是否正常 | 硬件调试 |

## 🎓 教学目标

通过本教程，学生能够：

1. **理解AI基础概念**
   - 什么是智能体
   - AI如何学习和工作
   - 人与AI的交互方式

2. **培养编程思维**
   - 明确的指令和规则
   - 条件判断（如果...那么...）
   - 调试和问题解决

3. **实践应用能力**
   - 创建自己的AI助手
   - 测试和验证功能
   - 发现和解决问题

4. **环保意识培养**（垃圾分类助手）
   - 认识垃圾分类
   - 理解环保的重要性
   - 在实践中学习

## 📊 教学建议

### 课堂组织

**第1课时：理论+猪猪侠**
- 10分钟：介绍什么是AI和智能体
- 20分钟：跟着教程创建猪猪侠
- 10分钟：测试和分享

**第2课时：垃圾分类助手**
- 10分钟：垃圾分类知识
- 25分钟：创建垃圾分类助手
- 5分钟：测试验证

**第3课时：自由创作**
- 让学生设计自己的智能体
- 分享和展示
- 总结学习

### 评价标准

- ⭐ 能够独立完成猪猪侠创建
- ⭐⭐ 能够理解垃圾分类规则
- ⭐⭐⭐ 能够成功创建垃圾分类助手
- ⭐⭐⭐⭐ 能够自己设计新的智能体
- ⭐⭐⭐⭐⭐ 能够帮助其他同学
