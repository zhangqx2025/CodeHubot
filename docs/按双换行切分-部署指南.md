```sql
    id,
```

```sql
-- 查看某个文档的文本块

    chunk_index,

    char_count,
```

## 💡 使用建议

### 适合使用"按段落（双换行）"的场景

✅ **推荐场景**：
1. **Markdown 文档** - 段落之间自然有双换行
2. **技术文档** - 通常段落分明
3. **教材、课本** - 结构化清晰
4. **邮件、报告** - 段落独立

❌ **不推荐场景**：
1. **无段落分隔的长文本** - 建议用"固定大小"
2. **代码文件** - 考虑其他切分方式
3. **对话记录** - 考虑"按句子"

### 优化建议

如果文档结构特殊，可以预处理：

```bash
# 示例：为没有双换行的文档添加双换行
# 在上传前，在每个句号后添加双换行

sed 's/。/。\n\n/g' input.txt > output.txt
```
